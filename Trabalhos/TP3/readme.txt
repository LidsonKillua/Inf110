Arquivos do PVANet:

- pnm-images.zip: imagens usadas no enunciado e outras no formato pnm (devem ser colocadas na mesma pasta do programa)

- TP3codbase.cpp: código base que lê uma imagem do arquivo chamado "imagem.pnm", escurece a imagem e a grava no arquivo "novaimagem.pnm"

- TP3codbaseb.cpp: o mesmo que o anterior, mas pergunta o nome do arquivo contendo a imagem

Obs.:
1- ao executar o código de TP3codbase.cpp deve haver na mesma pasta um arquivo chamado "imagem.pnm" com a imagem que deseja manipular,
pois o código só lê esse arquivo com esse nome. Caso contrário uma mensagem de erro será exibida.
2- o código TP3codbaseb.cpp já faz uma das tarefas do enunciado, que é perguntar o nome de um arquivo e ler a imagem daquele arquivo,
em vez de sempre de "imagem.pnm". O nome é armazenado em uma variável vetor de char, comumente chamada de string, que é assunto que
ainda será visto na disciplina (por isso já fiz esta parte).
